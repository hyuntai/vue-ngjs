# Advanced

  • Value 항목을 제외한 나머지 항목들은 콤보박스로 사전 정의된 항목 선택(화면 참고)
  • IF 일 경우 Value에는 4자리 이햐의 숫자만 입력 가능(Validation Check)
  • THEN 일 경우 Value에는 800byte 문자열 입력
  • Use Flag를 체크하면 ng_job 에 새로 추가한 post_flag = '1' (swpdb에 우선 반영)
  • 개별 라인을 추가/삭제할 수 있는 기능(버튼) 필요

# Resource Manager

  • Resource Lock 관리
