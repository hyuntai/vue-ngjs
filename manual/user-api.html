# APPENDIX


# User API

사용자 프로그램에서 스케줄링 중인 Job 조회 및 수행을 할 수 있도록 사용자 API 를 제공한다.

  • 헤더
    • pcp.h
  • 라이브러리
    • libpcp.so
  • API
    • pcpGetTodayInfo() : Today Job 정보 조회
    • pcpSetJobStatus() : Job 상태 변경
    • pcpSetJobAction() : Job Action 수행
    • pcpPubCond() : Condition 발행



# pcpGetTodayInfo() API

Today Job 정보 조회

# ProtoType

int pcpGetTodayInfo (
        const char *pzToday,
        const char *pzRscNm,
        const char *pzJobNm,
        const char *pzAppNm,
        const char *pzGrpNm,
        PCP_TODAY_INFO ** ppsTodayInfo,
        int nTimeoutMs);

# Parameters

[in] pzToday : Today(YYYYMMDD)
[in] pzRscNm : Resource 이름
[in] pzJobNm : Job 이름
[in] pzAppNm : Application 그룹 이름
[in] pzGrpNm : Group 그룹 이름
[out] ppsTodayInfo : Today Job 정보
[in] nTimeoutMs : Timeout (milliseconds)

# Description

입력받은 인자를 조건으로 Today Job 을 조회 한다. pzRscNm, pzJobNm, pzAppNm, pzGrpNm 이 NULL 일 경우 그 값은 조건에서 제외된다. 조회 내용은 ppsTodayInfo 변수에 저장된다.
ppsTodayInfo 변수를 사용한 뒤, 메모리 해지를 위해 pcpRelTodayInfo() API 를 수행 해야한다.

  • 사용 구조체
typedef struct {
    char zToday       [  8+1];
    int  nOrderId     ;
    int  nRscId       ;
    int  nJobId       ;
    char zRscNm       [ 64+1];
    char zJobNm       [ 64+1];
    char zApplication [ 64+1];
    char zGroupNm     [ 64+1];
    char zExecUser    [ 64+1];
    char zIsConfirm   [  1+1];
    char zTimeFrom    [  4+1];
    char zTimeTo      [  4+1];
    int  nDurDay      ;
    char zCyType      [  1+1];
    char zCyInterval  [  4+1];
    int  nCyMax       ;
    int  nCounter     ;
    int  nStatus      ;
    int  nAction      ;
    int  nPid         ;
    char zFirstTime   [ 14+1];
    char zStartTime   [ 14+1];
    char zEndTime     [ 14+1];
    char zStopFlag    [  1+1];
    char zSvrNm       [ 64+1];
} PCP_TODAY_JOB_INFO;

typedef struct {
    char zToday     [  8+1];
    int  nDelCnt    ;
    int  nCmplCnt   ;
    int  nErrCnt    ;
    int  nRunCnt    ;
    int  nNotRunCnt ;
    int  nExpCnt    ;
    int  nHoldCnt   ;
    int  nTotalCnt  ;
    PCP_TODAY_JOB_INFO *psJobInfo;
} PCP_TODAY_INFO;

# Return value

0: 성공
< 0: 실패

# Example

int i    = 0;
int nRet = 0;
PCP_TODAY_INFO *psTodayInfo = NULL;

// Today : 20200224
// Resource : MyResource#1
// Applicaion : NULL  // 검색 조건에서 제외됨.
// Group : MyGrp
nRet = pcpGetTodayInfo("20200224", "MyResource#1", NULL, "MyGrp", &psTodayInfo, 10000);
if (nRet != 0) {
    // error
}

printf("total count = %d\n", psTodayInfo->nTotalCnt);
for (i=0; i<psTodayInfo->nTotalCnt; i++) {
    PCP_TODAY_JOB_INFO *psJobInfo = (psTodayInfo->psJobInfo +i);
    printf("    %05d.%08d\n" ,i ,psJobInfo->nOrderId);
}

pcpRelTodayInfo(psTodayInfo); // psTodayInfo memory release

# pcpRelTodayInfo() API

Today Job 메모리 해제

# ProtoType

int pcpRelTodayInfo (PCP_TODAY_INFO * psTodayInfo);

# Parameters

[in] psTodayInfo Today Job 정보

# Return value

0 성공

# Description

pcpGetTodayInfo() API 에 의해 할당된 메모리를 해지한다. 메모리를 해지 하지 않으면 메모리 누수가 발생한다.

# Example

int nRet = 0;
PCP_TODAY_INFO *psTodayInfo = NULL;
nRet = pcpGetTodayInfo("20200224", "MyResource#1", NULL, "MyGrp", &psTodayInfo, 10000);
if (nRet != 0) {
    // error
}

pcpRelTodayInfo(psTodayInfo);

# pcpSetJobStatus() API

Job 상태를 변경 한다.

# ProtoType

int pcpSetJobStatus(
        const char *pzToday,
        int nOrderId,
        int nJobStatus,
        int nTimeoutMs)

# Parameters

[in] pzToday : Today(YYYYMMDD)
[in] nOrderId : Order ID
[in] nJobStatus : 변경 할 Job 상태
[in] nTimeOutMs : Timeout (milliseconds)

# Return value

0 성공
<0 실패

# Description

Job 상태를 변경 한다.

  • 변경 가능한 Job 상태
    • JOB_ST_NOTRUNNING

# Example

// Today : 20200224
// OrderId : 2021
// JobStatus : JOB_ST_NOTRUNNING (Not Running)
nRet = pcpSetJobStatus("20200224", 2021, JOB_ST_NOTRUNNING, 10000); // not run
if (nRet < 0) {
    //error
}

# pcpSetJobAction() API

Job Action을 수행 한다.

# ProtoType

int pcpSetJobAction (
        const char *pzToday, 
        int nOrderId, 
        int nJobAction, 
        int nTimeoutMs);

# Parameters

[in] pzToday : Today(YYYYMMDD)
[in] nOrderId : Order ID
[in] nJobAction : 요청할 Job Action
[in] nTimeOutMs : Timeout (milliseconds)

# Return value

0 성공
<0 실패

# Description

Job Action을 수행 한다.

  • 수행 가능한 Action
    • JOB_AT_RELEASE : Hold 된 Job 경우 Release 요청
    • JOB_AT_RERUN : Job 재수행 요청
    • JOB_AT_RUNNOW : Job 즉시 수행 요청
    • JOB_AT_HOLD : Hold 요청
    • JOB_AT_FORCED_OK : Job 강제 Complete 요청

# Example

// Today : 20200224
// OrderId : 2021
// JobStatus : JOB_AT_HOLD (Hold)
nRet = pcpSetJobAction("20200224", 2021, JOB_AT_HOLD, 10000); // Hold 수행  
if (nRet < 0) {
    //error
}

# pcpPubCond() API

Condition 을 발행 한다.

# ProtoType

int pcpPubCond (
        const char* pzToday, 
        const char *pzCondName, 
        int nTimeoutMs);

# Parameters

[in] pzToday : Today(YYYYMMDD)
[in] pzCondName : Condtition name
[in] nTimeoutMs : Timeout (milliseconds)

# Return value

0 성공
<0 실패

# Description

특정 날짜에 Condtition 을 발행 한다.

# Example

int nRet = 0;

// Today : 2020224
// Condition : MyJob#1_OK
nRet = pcpPubCond("20200224", "MyJob#1_OK", 10000);
if (nRet < 0) {
    // error
}
