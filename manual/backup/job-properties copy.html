# Job Type

Dummy

작업이 실행되지 않으나 여러 Workflow를 시작하거나 모을 때 형식적으로 등록한다.

Command

Argument 항목에 정의된 명령을 Agent에 전달한다.

Query

등록된 SQL을 실행하여 지정한 Format으로 파일로 저장한다.

File

파일을 AP에서 Batch서버(Upload)로 Batch에서 AP서버(Download)로 Copy한다.

# Cyclic

  • 주기적으로 설정한 횟수만큼 자동 실행되는 유형
    • Start : Interval을 Job 의 시작시점을 기준으로 계산한다.
    • End : Interval을 Job 의 종료시점을 기준으로 계산한다.
    • Cron : 등록된 시간에 실행함

# Duration

  • 실행되지 않은 Job은 설정한 Duration(일)이 지나면 Expire 처리함
    • 설정 범위 : 0~99, 0은 max 값으로 간주

Hold, Error, Running, Unknown 상태인 Job은 Expire 대상에서 제외됨

# Node

  • 명령을 실행할 Node(Agent)를 입력한다.
  • 등록된 Node를 선택하면 할당된 자원만큼 동시에 작업을 실행할 수 있다.
    • 작업당 필요한 자원의 양은 Required에 입력한다.
  • hostname을 직접 입력가능

# Owner

  • 작업실행계정
    • Agent 는 해당 계정으로 Switch 해서 작업을 실행한다.

# Status

  • Red: Error
  • Green: Complete(OK)
  • Gray: NotRun
