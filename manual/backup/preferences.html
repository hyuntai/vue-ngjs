# Preferences

  • Application 설정 화면
  • General
    • Condition Name Rule : Job Properties 저장 시점에 Out-Condition 이 존재하지 않으면 규칙에 따라 자동으로 생성된다.
    • Learn More(URL) : 도움말 페이지 URL
    • Debug Log : Debug Log On/Off
  • Notify PopUp
    • Display Time(sec) : Alarm창 Fade out time. 값이 0 이면 사라지지 않음.
    • Label Size : Alarm창의 크기 설정
  • Job Statics
    • Static Day : Job 통계화면 조회 기간
  • Server Monitoring
    • Interval(sec) : 서버의 Data를 Auto Refresh Interval
    • Tran Time Check(sec) : 마지막 Hotbeat 이 도착한 시간으로 부터 설정한 시간만큼 차이가 나면 모니터링 대상의 색상이 변경된다.
  • Job Log
    • Log Interval(sec) : Job Log의 Auto Refresh 주기
    • Event Interval(sec) : Job Event Log의 Auto Refresh 주기
  • Job 상태 별 색상 및 그룹의 배경색상 등을 지정한다.
