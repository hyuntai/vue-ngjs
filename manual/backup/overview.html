# Overview

  • PowerControl-Plus(이하 PCP)는 Digital Business 자동화를 위한 솔루션 입니다. PCP는 배치 작업이 정확한 시간과 순서로 수행됨을 보장하기 위해 이벤트 방식의 스케쥴링과 다양한 실행 조건을 제공합니다.

    • Batch Flow Chart를 포함한 직관적인 GUI
    • 실시간 모니터링용 Dashboard 와 System Event Log
    • 주요 이벤트 메세지 Push
    • 사용자 권한 관리 및 Audit Logging
    • 간단한 Job Type 설정만으로도 파일을 생성하기만 하면 복수의 서버로 전송 가능
    • 유연한 Workflow 설계를 위한 다양한 API 제공

  • PCP는 안정적인 서비스 운용을 위해 다양한 서버 구성을 지원합니다.

    • 실시간 Data Replication을 지원하여 HA(Active-Standby)구성이 가능합니다.
    • 2Node-Clustering(Active/Active)을 지원합니다.

# 주요 구성 요소

  • Server & Agent

    • PCP는 Batch Job들을 스케쥴링하고 작업을 지시하는 서버와 작업을 수행하고 결과를 전송하는 Agent로 구성됩니다.
    • 서버는 다른 원격 서버와 Workflow를 연동(Clustering)하도록 구성할 수 있습니다.

  • Resource

    • 리소스는 Job을 포함하는 논리적 그룹이며, 리소스 명은 전체 서버에서 유일하게 정의합니다.
    • PCP는 서버 영역(Repository)에서 Job을 통합 관리하며, 리소스는 Job 편집 위해 권한(Lock)을 획득하는 최소 단위입니다.
    • 리소스 단위로 준비작업(New day procedure) 시간과 유형(Auto/Manual)을 지정할 수 있습니다.
    • 일반적으로 단위 업무 혹은 팀 단위로 할당합니다.

  • Job

    • Job은 서버에서 스케쥴링 대상이 되는 개별 작업입니다.
    • Type, Condition, Schedule 등의 지정을 통해 다양한 Workflow를 구성할 수 있습니다.
    • PCP는 Job의 상태, 속성, 로그, 통계 조회 및 사용자 Action 전달을 위한 훌륭한 GUI를 제공합니다.

  • Condition

    • Job 실행을 결정하는 선행 조건을 의미합니다.
    • 일반적으로는 Job의 In/Out Condition을 의미하며, Workflow Path를 결정합니다.
    • Job 실행 조건으로 시간이나, Calendar, Cyclic, User Confirm, API등을 지정할 수 있습니다.
